Rigtools

Requirements: Chromebook version under 128.

Method

1. First click the first button below to copy the link, then paste it in a new tab and go back here.

2. Close the Devtools Tab then copy and paste the second link.

3. Click the "Network" Tab inside the new DevTools window.

4. Underneath "Name" press the gray check box 2 times.

5. Then click GForms and you can disable any extension.

ExtHang3r

Requirements: A Chromebook.

Method

1. Press the button to copy the link, then open it in a new tab.

2. Choose the extension you want to disable.

3. Press "Hang Extension", and after the process completes, press "Kill [Extension Name]".

Additional notes: Try lowering the iframes if your Chromebook crashes (e.g., to 3500 iframes).

BlomeVM

Description: BlobeVM is a virtual machine that can bypass school restrictions. Fast and easy to use.

Method

1. Go to GitHub Codespaces and choose the "blank" template.

2. Click the second button on this page to copy the command.

3. Paste it into the terminal and press enter.

4. Wait for the installation (4-20 minutes).

5. Once it's done, go to the "Ports" tab and click the globe icon next to the link to open the VM.

Incognito Exploit

Requirements: ChromeOS version under 128+ and `chrome://flags` not blocked.

Method

1. Copy the appropriate flag link for your Chrome version.

2. Paste the link into your browser, enable the flag, and restart.

3. Go to your Wi-Fi network settings, and set a custom name server using the IP address from the button below.

4. A "Sign in to network" notification will appear. Click it, then press CTRL+T to open an incognito tab.

Microsoft Labs VM

A simple VM for basic browsing.

Method

1. Sign into your Microsoft account.

2. Click the button below to go to Microsoft Labs.

3. Click "Launch VM Mode" and enter the password when prompted.

NOTE: Many websites are blocked. For a more capable VM, use BlomeVM.

ExtPrint3r

Works on any version of ChromeOS. Effect is temporary (5-10 minutes).

Method

1. Open the data URL from the button below.

2. In its settings, set the iframes to max and show all extensions.

3. Go to "The Expl0it" tab and select the extension you want to disable.
